## Local Setup

### Dev environment setup

Install node + npm (linux)

```
https://gist.github.com/dnavarrom/0b7c1e23d381269aa0a2870d51ff37b9
```

### Install grunt and build dependencies

```
npm install grunt grunt-cli -g
npm install

```

### Run local

```
grunt serve

```

open http://localhost:9000
